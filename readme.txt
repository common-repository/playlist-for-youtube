== Description ==

Display Youtube playlist on your webiste. This is the free version.

The pro version offer many settings :
- template choice
- color and text options
- show/hide title and description
- cache system for optimization

Some demos here of Pro version: [https://www.info-d-74.com/playlist-youtube-plugin-wordpress-demos/](https://www.info-d-74.com/playlist-youtube-plugin-wordpress-demos/)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Playlists Youtube => Settings to set your Youtube API key (get it here : https://console.developers.google.com/home)
4. Go Playlists Youtube to create your first playlist


== Frequently Asked Questions ==

1. The plugin work with any playlist Youtube ?

No, only with playlists that are public (most of the playlists)

2. What is needed to configure the plugin ?

Only 2 things : ask an API key for Youtube API and IDs of the playlist to display
